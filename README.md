"# drive-dork-bulk-checker" 
## Usage

1. Edit the `domains` variable in `dork_search.sh` (space-separated domains).
2. Run the script in a Bash environment (Git Bash, WSL):
   chmod +x dork_search.sh
   ./dork_search.sh
3. Review results printed in the terminal and in `all_results.txt`.
